Number Class

Trong những bài học trước ta đã tìm hiểu về cấc kiểu primitive type. Hôm nay ta sẽ đi sâu vào từng thằng một

Mục đích đó là cần xem và làm để biết Java nó support những cái gì, sau này có google thì cũng dễ.

Thông thường, ta chỉ làm việc với các kiểu dữ liệu nguyên thủy, primitive, tuy nhiên có một vài trường hợp ta cần làm việc với các kiểu number dưới dạng object.

Java cung cấp một đống wrapper classes để mà hỗ trợ cái này. Tương ứng với: Integer, Long, Byte, Double Float, Short. Tất cả tụi này đều là con của abstract class Number

Các object của các wrapper class chứa hoặc bao bọc các dữ liệu primitive của chính nó.

Việc chuyển đổi các kiểu dữ liệu nguyên thủy primitive thành đối tượng được gọi là boxing, và được thực hiện bởi trình biên dịch. Vì vậy nên ta chỉ cần truyền giá trị primitive cho constructor của Wrapper Class

Và object Wrapper sẽ được convert ngược lại thành primitive type. Quá trình này được gọi là unboxing. Class Number là một phần của java.lang.package.

public class Test{
    public static void main(String args[]){
        Integer x = 5; // boxes int to an Integer object
        x = x + 10; // unboxes the Integer to a int
        System.out.println(x);
    }
}

When x is assigned an integer value, the compiler boxes the integer because x is integer object. Later, x is unboxed so that they can be added as an integer.

compareTo(_number) Method này nhằm so sánh cái thằng number hiện tại với _number kia. Giá trị trả về là 0 nếu 2 thằng đó bằng nhau, 1 nếu object hiện tại lớn hơn và -1 nếu nó bé hơn. Dùng cái này trong sort number cũng được
equals() Dùng để so sánh 2 object có phải là 1 không, giá trị là true / false. So sánh này là về mặt giá trị, có lẽ do nó override lại cái method
valueOf() Trả lại một object gì đó bằng cách convert dữ liệu. KIểu như truyền vào String mà nó đổi thành Integer vậy
toString() Chuyển đổi kiểu number thành String.
parseInt() Chuyển kiểu dữ liệu hiện tại thành kiểu Integer.
Math - abs Trả lại giá trị absolute của object, = -a
Math - ceil Làm tròn lên. Tức là có 100.55 thì sau khi chạy Math.ceil(b) nó trả về 101.
Math - fllor Làm tròn xuống. Cái trên nếu gọi Math.floor(100.55) thì nó trả về 100.
Math - rint (Round to Integer) Nó sẽ làm tròn cái giá trị đó, tới cái giá trị integer gần nhất. Nó như round vậy nhưng mà chỉ lấy int. Và kết quả trả về là số long. 100.0
Math - round Cái này như cái trên kia, chỉ là kết quả của nó trả về là integer / long. Một điểm cần ghi nhớ ở đây. Math.rint(100.500) thì trả về 100, trong khi đó, Math.round(100.500) thì trả về 101.
Math - min, max Lấy giá trị nhỏ nhất / lớn nhất giữa 2 số.
Math - exp Lấy logarithms, cái này là gì quên rồi :(((
Math - pow Lấy số mũ. Math.pow(10, 2) = 10 ^ 2
Math - sqrt Cái này thì quen rồi, lấy căn bậc 2
Math - sin, cos, atan, acos Lưu ý, cái này tham số truyền vào của nó là radians nhé. Vì vậy cần gọi method Math.toRadians(degree) để chuyển cái angle về đơn vị radians trước khi tính toán. Tương tự, ta cũng có thể gọi toRadians để convert ngược lại.
Math - random Method nà khác hữu ích. Nó random 1 giá trị trong khoảng 0.0 --> 1.0. Vậy, muốn random trong một khoảng thì sao. Giả sử ta có giá trị min = 100, max = 200. Khi đó, Math.random trả về 1 giá trị từ 0.0 --> 1.0. với giá trị 0 --> = min_value, với giá trị 1 = max value.

Sử dụng công thức này để đạt được điều đó.

Math.floor(Math.random() * (max_value - min_value)) + min_value;

Khi random == 0 --> exp = min. Khi random == 1, exp = max - min + min = max.

Array Class
