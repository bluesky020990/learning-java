String In Java

String thì ai cũng biết rồi, vì cái ví dụ đầu tiên là dòng Hello World của nó.

Nhưng để hiểu sâu hơn về nó thì ta cần xem xét đấy.

Bản chất

String được define trong java.lang package. Nó không phải là primary data như mấy cái thằng int, long, double đâu nhé. Vậy nó là Object à? Cũng có thể phải, mà cũng có thể ko.

String được khai báo bằng 2 cách, dùng toán tử new String() hay dùng dấu "";

Nếu dùng dấu "" để khởi tạo, khi đó giá trị của nó sẽ là một giá trị trong String Pool. Có vẻ như Java biết nó sẽ được sử dụng nhiều nên ưu ái cho nó cái này. Khi một String được khởi tạo, String pool kiểm tra xem nó có giá trị nào như thế ko, nếu có thì trả về ( luôn là địa chỉ tham chiếu nhé) còn không thì nó sẽ tạo ra một đối tượng mới, và lưu trữ nó trong String Pool trước khi trả về.

Nếu sử dụng toán tử new thì ngược lại, nó được xem như một Object, NHƯNG nó sẽ KHÔNG được đưa vào String Pool. Nếu muốn đưa nó vào String pool ta phải sử dụng câu lệnh intern().

Palindrome

Palindrome là chuỗi mà việc đọc từ trái sang hay phải sang thì cũng đều như nhau.

Cách check một String có phải là palindrome hay ko rất đơn giản, chuyển nó thành array với mỗi element là 1 char, revert array đó và so sánh nó với cái ban đầu, equal thì chứng tỏ là true.

Để làm điều này ta có thể sử dụng chức năng revert của String Builder

Còn ko thì cứ for each từ 0 -> n / 2 và từ n -> n /2 , nếu bất cứ trường hợp nào mà f1 != f2 thì ta trả lại false thôi.
