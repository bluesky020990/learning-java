Access Modifier in Java

Ở lesson này ta sẽ đi sâu vào tìm hiểu phần Access Modifier.

Trong Java, có 4 cấp độ access bao gồm

  • private - Chỉ thuộc class đó, các class con ko được truy xuất
  • protected - Chỉ class đó và các class trong cùng packet mới có thể thấy
  • public - Khai báo này thì tất cả đều thấy các properties và method.
  • default - Nằm giữa protected và public :

Private Access Modifier in Java Inheritance

A sub class can take all the attributes of a super class which is having public and protected access, it cannot access a private members of the parent class, as the private variable belongs to that class itself. In those cases the we can access those private members of the parent class using a public access methods (getters & setters). Lets change the Cycle class to have private members.

Subclass có thể kế thừa toàn bộ thuộc tính của super class mà được khai báo là public và protected. Tuy nhiên những private properties thì nó không kế thừa được.

Nhưng thường thì chúng ta có thể sử dụng các getter setter để có thể truy cập vào các thuộc tính bên trong nó.
