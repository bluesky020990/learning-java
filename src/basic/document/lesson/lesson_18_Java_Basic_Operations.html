Java Basic Operators

Java cung cấp một bộ các toán tử rất phong phú. Dựa vào chức năng ta có thể chia nó thành các loại sau:

  • Arithmetic Operators
  • Relational Operators
  • Bitwise Operators
  • Logical Operators
  • Assignment Operators
  • Míc Operators

The Arithmetic Operators - Toán tử số học

Nói về thằng này thì ta sẽ nhớ tới các toán tử cơ bản như +, -, *, /, ++, --, %

a + b

a - b

a * b

a / b

a++ => a = a + 1

a-- => a = a - 1

a % b

The Relational Operators

Đây là các toán tử quan hệ, bao gồm : ==, != , >, <, >=, <=

a == b

a != b

a > b

a < b

a >= b

a <= b

The Bitwise Operators

Tới giờ vẫn chưa hiểu cái Bitwise này để làm gì nữa.

The Logical Operators

Cái này thì bao gồm: &&, ||, !

a && == true if a == true and b == true (cả 2 đều đúng, sai 1 cái coi như sai)

a || b == true if a == true or b == true (một trong 2 đúng là được)

! a == true if a == false (phép nghịch đảo)

The Assignment Operators

Gồm =, +=; -=; *=; /=, %= (C = C % A)

a = b;

a += b --> a = a + b

a -= b --> a = a - b

a *= b --> a = a * b

a /= b --> a = a / b

a % b --> a = a % b

Miscellaneous Operators

Quan trọng nhất và cơ bản nhất bao gồm

Toán tử điều kiện ?

variable x = (expression) ? value if true : value if false

instanceof Operator

Toán tử này chỉ dùng với các biến tham chiếu (object/reference variable). Nó dùng để kiểm tra xem một object có phải là một kiểu đặc biệt (class type or interface type) hay ko.

objectA.instance(CheckClass.class) == true | false
