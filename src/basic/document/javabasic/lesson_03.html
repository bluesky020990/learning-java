How to create Object in Java

Thử nghĩ xem có bao nhiêu cách để tạo Object nào

Xem ví dụ trong file JavaLesson03

Sử dụng toán tử new.

JavaLesson03 demoClass = new JavaLesson03();
demoClass.sayHello();

Sử dụng class name

Class clazz = Class.forName("practice.basic.lesson03.JavaLesson03");
JavaLesson03 demo = (JavaLesson03) clazz.newInstance();
demo.sayHello();

Chú ý cái giá trị này phải truyền đúng nhé practice.basic.lesson03.JavaLesson03

Sử dụng Class Loader và newInstance() Method

ClassLoader cl = JavaLesson03.class.getClassLoader();
JavaLesson03 t = (JavaLesson03)cl.loadClass("practice.basic.lesson03.JavaLesson03").newInstance();

Using Object Cloning – clone() method

JavaLesson03 demoClass = new JavaLesson03("The object create by new operator");
try {
    JavaLesson03 cloneObject = (JavaLesson03) demoClass.clone();
    cloneObject.setMessage("The object create by clone method");
    cloneObject.sayHello();
} catch (CloneNotSupportedException e) {
    e.printStackTrace();
}

Sử dụng phương thức này thì nhớ overwrite clone method cho class này nhé.

@Override
protected Object clone() throws CloneNotSupportedException {
    return new JavaLesson03(this.getMessage());
}

Object Class trong Java

Thằng này là cha của tất cả các class trong Java.

The Object class is the Parent class of all the classes in Java. In other words java.lang.Object class is the root of the Class Hierarchy. For each and every class which we create this class will be acting as the Super class and that is the reason why we are able to use hashCode(), toString(), wait() etc.. methods directly which actually belongs to the Object class

All the class which is declared will be extending the class Object, even though if we didn’t specify it directly it will be happening implicitly. This will be taken care by the JVM itself, but it doesn’t apply to Interfaces, as a Java Interface cannot extend a class.
